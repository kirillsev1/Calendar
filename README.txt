Описание приложения "Календарь настроения":

1)Язык программирования - Python.
2)Среда разработки - PyCharm.
3)Функционал:
->Настроение отмечается один раз в день.
->Возможность просматривать истрию отметок.
->Отметки настроения созраняются между запусками.
->Для удобства представления информации об истории отметок есть встроенная диограма.



Важно! Чтобы запустить программу, нужно скачать из Интернета интегрированную среду для языка 
программирования Python (PyCharm или Python IDLE(3.10 64-bit)).

Инструкция по установке сред:

1. Чтобы скачать PyCharm надо перейти на их официальный сайт(https://www.jetbrains.com/ru-ru/pycharm/)

-> В правом верхнем углу будет кнопка скачать, нажимаем на нее
-> Отрывается окно с выбором ситемы (Windows, macOS, Linux)
-> Скачиваем версию Community
-> Нажимаем скачать
-> После загрузки файла .exe, открываем его и устанавливаем на компьютер.


2. Чтобы скачать IDLE(Python) необходимо:

-> Зайти на официальный сайт Python(https://www.python.org/)
-> Навести курсор на кнопку Downloads и нажать на кнопку Python 3.10
-> Затем надо запустить файл .exe и установить программу.



3. После установки PyCharm IDLE(Python 3.10 64-bit):

-> Разархивировать все данные из zip файла в условную папку
-> Открыть паку "Календарь", и найти там файл "main.py"


Запуск программы осуществляется через одну из описанных выше сред.


					Если запускать через PyCharm:

-> Нажать правой клавишей мышки на файл "main.py" и выбрать пункт "Edit with PyCharm Community Edition"
-> Установка Библиотек (tkinter, datetime, matplotlib.pyplot).
Для этого открываем командную строку и пишем там "py -m pip install 'Название библиотеки'"
-> Когда весть проект загрузится надо нажать на зеленый треугольник в правом верхнем углу
-> Открывается окно "Запись настроения"- можно приступать к работе.
-> Если надо остановить программу, надо нажать на красный квадрат в правом верхнем углу,
который появится там после запуска программы.


				Если запускать через IDLE(Python 3.10 64-bit):

-> Сначала надо извлечь все данные из zip файла
-> Потом надо открыть IDLE(Python 3.10 64-bit), это можно сделать через поисковую строку системы компьютера
-> Далее в левом верхнем углу надо нажать на кнопку "file" и выбрать пункт "open"
-> Теперь надо найти файлы, которые ранее находились в zip файле, выбрать "main.py" и нажать кнопку "Открыть"
-> Установка Библиотек (tkinter, datetime, matplotlib.pyplot).
Для этого открываем командную строку и пишем там "py -m pip install 'Название библиотеки'"
-> Теперь надо нажать на кнопку "run" и выбрать "Run Module"
-> Открывается окно "Запись настроения"- можно приступать к работе.


Приятного использования!
